## Generation Locks (Optional)

To prevent cross-instance duplicate level-map generation, the API can use a small lock table. Create it once in your Supabase database:

```sql
create table if not exists generation_locks (
  user_id uuid not null,
  subject text not null,
  created_at timestamptz not null default now(),
  primary key (user_id, subject)
);
```

If this table is not present, the app falls back to in-process locking and client-side backoff.
